# slot submodule

This submodule manages deployment slots (`Microsoft.Web/sites/slots`) for an Azure App Service site.

It can be used independently to create and configure a deployment slot on an existing site, including:
- The slot resource itself
- Slot app settings
- Slot connection strings
- Slot storage account mounts
- Slot FTP and SCM publishing credential policies
- Slot locks, role assignments, and private endpoints
