# Application Stack Configuration

This example deploys a Linux Web App with a custom application stack configured via the `site_config.application_stack` variable.

It demonstrates how to specify runtime frameworks such as .NET, Node.js, Python, Java, PHP, or PowerShell for your App Service. The application stack settings control which language runtime is used to serve your application.

The example uses `kind = "webapp"` and `os_type = "Linux"`.
